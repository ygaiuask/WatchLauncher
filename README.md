# Forigon

Launcher for Android watches. Works on full Android watches and Wear OS (not using wear os compose, so not integrated, hence shouldn't be a better experience than default launcher).

## Features

- List or bubble cloud app drawer
- Virtual rotary bezel with haptic detents
- Quick settings panel (flashlight, DND, WiFi, Bluetooth)
- Search with fuzzy matching and transliteration
- Hide apps
- Configurable gestures (long-press or double-tap for options)
- Motion modes for low-power devices
- UI scaling for tiny screens

## Controls

| Input | Action |
|-------|--------|
| Tap watch face | App drawer |
| Long-press watch face | Settings |
| Swipe up | App drawer |
| Swipe down | Quick settings |
| Edge drag (clockwise/counter) | Scroll / zoom |
| D-pad up/down | Navigate |
