# WatchLauncher - 安卓手表桌面启动器

> 基于 [Forigon](https://github.com/mlm-games/forigon) 二次开发，专为 Android 8.1+ 智能手表优化

## 功能特性

- **应用抽屉**: 气泡云视图和列表视图，快速找到并启动应用
- **虚拟旋转表圈**: 边缘手势滑动，带触觉震动反馈
- **快速设置**: 手电筒、WiFi、蓝牙、亮度、勿扰模式一键开关
- **智能搜索**: 支持中文拼音模糊搜索，键盘自动弹出
- **应用管理**: 隐藏不常用应用，自定义应用名称
- **手势操作**: 上滑打开应用抽屉，下滑打开控制中心，长按进入设置
- **圆形/方形屏幕**: 自适应不同手表屏幕形状
- **省电模式**: 减少动画效果，延长手表续航
- **Material 3 UI**: 深色主题，现代视觉设计

## 系统要求

- Android 7.0 (API 24) 及以上
- 专为 Android 8.1 (API 27) 优化
- 适用于运行完整安卓系统的手表设备

## 快速开始

### 安装

1. 下载 APK 文件到手表
2. 使用文件管理器打开 APK 进行安装
3. 或通过 ADB 安装: `adb install WatchLauncher.apk`
4. 按 Home 键，选择"WatchLauncher"为默认启动器

### 基本操作

| 操作 | 功能 |
|------|------|
| 点击表盘 | 打开应用抽屉 |
| 长按表盘 | 打开设置 |
| 上滑 | 打开应用抽屉 |
| 下滑 | 打开控制中心 |
| 边缘旋转滑动 | 滚动/缩放 |
| 搜索框输入 | 查找应用 |

## 修改说明

本项目对原版 Forigon 进行了以下关键修改：

1. **Android 8.1 适配**: targetSdk 下调至 27，移除高版本专属 API
2. **中文优化**: 完善中文本地化，符合国内用户习惯
3. **兼容性增强**: 移除 API 28+ 特性标记，确保稳定运行
4. **构建简化**: 优化 Gradle 配置，降低编译环境要求
